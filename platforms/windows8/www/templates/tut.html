Welcome to Konectivity

Konectivity provides a professional platform for users to connect with people who share similar interests.

Easy to login

Users can use their LinkedIn account to log in to Konectivity or sign-up to create their own account.

Menu

Menu Bar provides navigation for users to access all the features provided by Konectivity

Potential Connections

Users can see other people who are using Konectivity to make new connections.

My Connection

After accepting connection requests, users become konectivity connections. Users can also send messages to their connections.

Event Page

By entering correct event ID provided by event planner, users can see details about the event.
